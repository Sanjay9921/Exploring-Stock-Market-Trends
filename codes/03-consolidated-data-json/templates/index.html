Consolidated Wealth Data API

This module exposes JSON endpoints over the unified portfolio database.

Live JSON endpoints

  • /api/accounts – all accounts
  • /api/holdings – all holdings
  • /api/holdings?acct_id=1 – holdings for account 1
  • /api/ohlcv?ticker=AAPL – OHLCV for AAPL

Export static JSON for Power BI

First generate static JSON snapshots (accounts.json and holdings.json) under codes/03-consolidated-data-json/data/json_exports/:

  • /api/export/json – create/update static JSON files

Serve static JSON over HTTP

After exporting, you can also consume the accounts snapshot via this static URL (usable with Power BI's Web connector):

  • /static-json/accounts – static accounts.json
